# Uni-Pinned Components

*Free — requires [Uni-Inspector](uni-inspector.md)*

A permanent shelf for the handful of components you reach for on nearly every GameObject — no
more hunting through **Add Component** for the same script every time.

- **Pin a component** — drag it out of the Inspector and drop it into the window.
- **Organize** — group pinned components into categories with their own tab strip.
- **Use it** — click a pinned entry to add it to the selected GameObject.
