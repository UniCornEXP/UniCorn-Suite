# Uni-Compare Components

*Free — requires [Uni-Inspector](uni-inspector.md)*

Drag two GameObjects in to instantly highlight what's different:

- **amber** = differs (→/← syncs it)
- **rose** = present on only one side (**Add** copies it across)

Expand any component for a field-by-field breakdown. Everything writes straight to the real
GameObject with full Undo support.
