# Troubleshooting

**Header buttons missing.**
A custom editor has replaced the default component header entirely, so the extra icons have
nowhere to attach.

**Uni-Preview: a dragged object shows nothing.**
It needs an animation, a particle system, a renderer, or a UI graphic somewhere in its hierarchy.

**"Group Selected" isn't in Uni-Hierarchy's right-click menu.**
Use the toolbar's Group icon instead — it checks your selection and tells you why it can't group
it if not.

**Pasted/restored the wrong thing.**
Ctrl/Cmd+Z — every write goes through Unity's Undo system.

**Does this send anything to a server?**
No. Every product only reads and writes your own project, locally.
